以下是Nyabot可用的lua函数：
Part1 动作类函数


CqAct:SetGroupAdmin(GroupID, UserId, Enable)函数：添加/撤销群管理员
GroupID（int参数）：群ID
UserID（int参数）：群内用户ID
Enable（bool参数）：为True则加管理员权限，为False则撤管理员权限
:::tip
执行此函数，需要机器人在群中拥有管理员权限
:::
CqAct:GroupMute(GroupID, UnMute)函数：开关全员禁言
GroupID（int参数）：要操作的群ID
Unmute（bool参数）：为True则开启，为False则关闭
:::tip
执行此函数，需要机器人在群中拥有管理员权限
:::
CqAct:GroupKick(GroupID, UserID, RejectAddRequest)函数:从群中踢出一位群成员
GroupID（int参数）：要操作的群id
UserID（int参数）：要踢出的用户id
RejectAddRequest（bool参数）设置为True则不允许此人再申请加群
:::tip
执行此函数，需要机器人在群中拥有管理员权限
:::
 CqAct:LeaveGroup(GroupID)函数：让机器人离开群
GroupID（int参数）：要操作的群id
CqAct:DismissGroup(GroupID int64)函数：解散群聊
GroupID（int参数）：要操作的群id
:::tip
执行此函数，需要机器人在群中拥有管理员权限
:::
