## 操作类函数

-----

### 添加/撤销群管理员

:::tip

执行此函数，需要机器人在群中拥有管理员权限

:::

#### 函数：`CqAct:SetGroupAdmin(GroupID, UserId, Enable)`

- GroupID（int 参数）：群 ID。
- UserID（int 参数）：群内用户 ID。
- Enable（bool 参数）：为 True 则加管理员权限，为 False 则撤管理员权限。

### 开关全员禁言

:::tip

执行此函数，需要机器人在群中拥有管理员权限

:::

#### 函数：`CqAct:GroupMute(GroupID, UnMute)`

- GroupID（int 参数）：要操作的群 ID。
- Unmute（bool 参数）：为 True 则开启，为 False 则关闭。

### 踢出群成员

:::tip

执行此函数，需要机器人在群中拥有管理员权限

:::

#### 函数：`CqAct:GroupKick(GroupID, UserID, RejectAddRequest)`

- GroupID（int 参数）：要操作的群 ID。
- UserID（int 参数）：要踢出的用户 ID。
- RejectAddRequest（bool 参数）设置为 True 则不允许此人再申请加群。

### 机器人离开群

#### 函数：`CqAct:LeaveGroup(GroupID)`

- GroupID（int 参数）：要操作的群 ID。

### 解散群聊

:::tip

执行此函数，需要机器人在群中拥有管理员权限

:::

#### 函数：`CqAct:DismissGroup(GroupID int64)`

- GroupID（int 参数）：要操作的群 ID。
