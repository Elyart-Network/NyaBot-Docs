# 🗨️ 测试

:::note

如果你确认你已经配置好你的bot，可以准备开始测试！

:::

## Lua 脚本测试

-----

### 1. 准备脚本

#### 将以下代码复制并保存在 `/运行目录/script` 下，并将文件命名为 `example.lua`。

```lua
  if Callback.CqCall.MessageData == "喵呜" then
    if Callback.CqCall.GroupID ~= 0 then
        CqMsg:Reply("喵~~~", Callback.CqCall.GroupID, false, Callback.CqCall.MessageID)
        end
    if Callback.CqCall.GroupID == 0 then
        CqMsg:Reply("喵~~~", Callback.CqCall.UserID, false, Callback.CqCall.MessageID)
        end
  end
```

### 2. 修改配置

#### 打开目录下的 `lua.ini` 文件, 检查是否存在以下片段，并修改为以下内容：

```ini
[example]
enable=true
script=example.lua
```

### 3. 运行脚本

#### 向你的 Bot 发送私聊消息 ”喵呜“，当然你也可以将bot拉入群聊，发送”喵呜“。

:::tip

如果你的 Bot 运行正常，你的 Bot 应该会回复你 ”喵~~~“。

恭喜你，你可以开始做更高级的开发了！

:::

:::caution

如果你的 Bot 没有回复任何内容，请检查你的连接与配置是否正确。

前往 [🔗连接](/docs/guide/gocqhttp-connect) 与 [🛠️配置](/docs/guide/config) 章节查看更多信息，前往 [🔍故障](/docs/guide/troubleshoot) 章节寻求故障排除与帮助。

:::
