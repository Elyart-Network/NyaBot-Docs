# 📝 介绍

## 关于 NyaBot

NyaBot 是一个基于 Go 语言的开源聊天机器人框架，支持多种聊天平台协议。

它具有高效、灵活、可扩展的特点。它提供了丰富的API和工具，可以帮助用户快速开发自己的聊天机器人应用。

## 目标

NyaBot致力于成为一个**快速、简单、轻量、强大、可扩展**的聊天机器人框架。

- **快速**。得益于 Go 语言的静态编译技术，NyaBot 的编译速度极快。

- **简单**。NyaBot 开箱即用，仅需要简单配置即可运行。

- **可扩展**。 NyaBot本身极具扩展性，并且原生支持 gRPC 协议，可跨语言开发插件。它还支持使用 Lua 动态脚本，JavaScript 动态脚本进行插件开发。

- **轻量与强大**。 NyaBot 使用 Go 语言进行开发，可实现低占用，高并发。
