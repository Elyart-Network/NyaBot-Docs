# ⚠️QQ风控相关

:::info

风控是**不可避免的正常情况**，您应当正确认识并了解如何应对风控
记住，害怕和逃避是解决不了问题的！

:::

:::info

风控和封禁**不是一个概念**，被风控并不代表被封禁！

:::

通过goCqhttp协议制作的QQ机器人，通常可能出现如下风控情况：
请查看登录错误的Code，并对照检查

附赠一些防封禁，防风控小技巧：
- 机器人发送信息尽量不包含敏感词
- 机器人发信息间隔2-4秒
- 不要经常用其他设备挤登录，频繁切换会封号或者风控
- 机器人账号的等级尽量大于24级
- 新注册7天的号不要登框架
- 机器人账号近期如果经常屏蔽或掉线，等48~72小时后再挂
- Q龄大的号不易风控
