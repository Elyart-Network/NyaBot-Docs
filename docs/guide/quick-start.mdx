# 🚀 开始

:::tip

NyaBot 的 **[GitHub](https://github.com/Elyart-Network/NyaBot/releases)** 内提供了所有发布版本，可从此处下载对应平台的可执行文件。

:::

### 启动NyaBot

<Tabs>
<TabItem value = "Windows" label = "Windows" default>
        <p>1. 打开您的命令提示符（它可以是 CMD, PowerShell 等）。</p>
        <p>2. 将您的命令提示符路径转到您存放 NyaBot 可执行文件的路径内。</p>
        <p>3. CMD: 直接执行 NyaBot.exe (前提是您没有更改程序名)：</p>
        <pre>NyaBot.exe</pre>
        <p>3. PowerShell : 执行 NyaBot.exe (前提是您没有更改程序名)：</p>
        <pre>./NyaBot.exe</pre>
</TabItem>
    <TabItem value= "Linux" label= "Linux">
        <p>1. 打开您的终端或 SSH 。</p>
        <p>2. 将您的终端转到您存放 NyaBot 可执行文件的路径内。</p>
        <p>3. 为 NyaBot 添加可执行权限：</p>
        <pre>chmod +x ./NyaBot</pre>
        <p>4. 执行 NyaBot：</p>
        <pre>./NyaBot</pre>
    </TabItem>
    <TabItem value= "MacOS" label= "MacOS">
        <p>1. 打开您的终端或 SSH 。</p>
        <p>2. 将您的终端转到您存放 NyaBot 可执行文件的路径内。</p>
        <p>3. 为 NyaBot 添加可执行权限：</p>
        <pre>chmod +x ./NyaBot</pre>
        <p>4. 执行 NyaBot：</p>
        <pre>./NyaBot</pre>
    </TabItem>
    <TabItem value= "Android" label= "Android">
        <p>1. 下载 <a href= "https://github.com/termux/termux-app/releases/">Termux</a> 或其他终端程序</p>
        <p>2. 将您的终端转到您存放 NyaBot 可执行文件的路径内。</p>
        <p>3. 为 NyaBot 添加可执行权限：</p>
        <pre>chmod +x ./NyaBot</pre>
        <p>4. 执行 NyaBot：</p>
        <pre>./NyaBot</pre>
    </TabItem>
</Tabs>
